Разработка и верификация алгоритмов для моделирования колебаний гармонической и ангармонической цепочек

Apr 11, 2026 5 min read

Цель работы: Разработать и описать численные алгоритмы для моделирования динамики одномерной цепочки связанных частиц.

Основное внимание уделяется реализации двух ключевых постановок:

  1. Гармоническая цепочка: алгоритм, точно воспроизводящий аналитическое решение в виде стоячих волн (собственных мод).
  2. Ангармоническая цепочка (задача Ферми–Пасты–Улама): алгоритм, позволяющий исследовать перераспределение энергии между модами из-за нелинейности.

Задание

В соответствии с описанием проекта 2.8 в учебном пособии [@medvedev_modeling_2010, с. 86–90] необходимо разработать алгоритмы для:

  1. инициализации системы частиц с заданными параметрами (m = 1, k = 1, d = 1);
  2. расчёта сил в двух режимах: линейном (гармоническом) и нелинейном (ангармоническом);
  3. численного интегрирования уравнений движения с использованием алгоритма Верле;
  4. задания начальных условий в виде собственных мод колебаний;
  5. верификации модели для гармонического случая;
  6. реализации дискретного преобразования Фурье (ДПФ) для анализа энергетического спектра.

Теоретическое введение

1. Модель гармонической цепочки

Рассматривается одномерная система из N точечных частиц массой m, соединённых одинаковыми пружинами жесткости k. Крайние частицы прикреплены к неподвижным стенкам. Положение равновесия i-й частицы: x_i = i · d. Введём смещение y_i от положения равновесия [@medvedev_modeling_2010, с. 87].

Уравнение движения для i-й частицы:

m · (d²y_i)/(dt²) = k · (y_{i+1} - 2y_i + y_{i-1}). (1)

Это линейная система, решениями которой являются стоячие волны. Собственные частоты ω_l для l-й моды определяются дисперсионным соотношением:

ω_l = 2 · √(k/m) · sin( l · π / (2 · (N + 1)) ). (2)

В безразмерных переменных (m = 1, k = 1, d = 1) формула упрощается.

2. Модель ангармонической цепочки (FPU)

Для учёта нелинейности в возвращающую силу пружины добавляется квадратичная поправка [@medvedev_modeling_2010, с. 88]. Сила, действующая на пружину, деформированную на величину x, принимает вид:

F = –k · x · (1 – α · x / d). (3)

Параметр α (безразмерный) определяет степень ангармонизма. При α > 0 сила растёт медленнее линейной при растяжении и быстрее — при сжатии. Такая нелинейность приводит к взаимодействию между различными собственными модами системы.

3. Дискретное преобразование Фурье

Для анализа распределения энергии по модам используется дискретное преобразование Фурье (ДПФ) [@medvedev_modeling_2010, с. 90]. Коэффициенты b_l (для синусного разложения) вычисляются по формуле:

b_l = (2/N) · Σ_{j=0}^{N-1} y(x_j) · sin( 2 · π · j · l / L ). (4)

Энергия, приходящаяся на l-ю моду, пропорциональна E_l ∝ b_l² + ω_l² · (db_l/dt)².

Выполнение лабораторной работы

1. Разработка алгоритмов

1.1. Алгоритм инициализации системы

Вход: N (число частиц), mode_number (номер моды l), amplitude (амплитуда A).
Выход: массивы y[i] и v[i].

Алгоритмы численного моделирования динамики цепочки частиц

1. Описание алгоритмов

1.1. Алгоритм инициализации системы

Вход: N, mode_number, amplitude.
Выход: y, v.

Функция InitializeSystem(N, mode_number, amplitude):
    m = 1; k = 1; d = 1; alpha = 0.0
    L = (N + 1) * d

    для i от 1 до N:
        x = i * d
        y[i] = amplitude * sin(mode_number * π * x / L)
        v[i] = 0.0

    вернуть y, v

##1.2. Алгоритм расчёта сил и ускорений

Вход: y, k, d, alpha.
Выход: a[i].

def ComputeAccelerations(y, k, d, alpha):
    N = len(y)
    a = [0.0] * N
    
    for i in range(N):
        delta_left = y[i] - (y[i-1] if i > 0 else 0)
        delta_right = (y[i+1] if i < N-1 else 0) - y[i]
        
        F_left = k * delta_left
        F_right = -k * delta_right
        
        if alpha > 0:
            F_left = F_left * (1 - alpha * delta_left / d)
            F_right = F_right * (1 - alpha * delta_right / d)
        
        F_total = F_right - F_left
        a[i] = F_total / 1.0  # m = 1
    
    return a

1.3. Алгоритм численного интегрирования (скоростной метод Верле)

Вход: y, v, a, dt.
Выход: y_new, v_new, a_new.

def VelocityVerlet(y, v, a, dt):
    N = len(y)
    y_new = [0.0] * N
    
    # 1. Обновление позиций
    for i in range(N):
        y_new[i] = y[i] + v[i] * dt + 0.5 * a[i] * dt**2
    
    # 2. Вычисление новых ускорений
    a_new = ComputeAccelerations(y_new, k=1, d=1, alpha=0.0)
    
    # 3. Обновление скоростей
    v_new = [0.0] * N
    for i in range(N):
        v_new[i] = v[i] + 0.5 * (a[i] + a_new[i]) * dt
    
    return y_new, v_new, a_new

1.4. Алгоритм спектрального анализа (ДПФ)

Вход: y.
Выход: E[l].

def ComputeModeEnergies(y):
    import math
    
    N = len(y)
    d = 1.0
    L = (N + 1) * d
    E = [0.0] * (N // 2)
    
    for l in range(1, N // 2 + 1):
        sum_sin = 0.0
        for j in range(1, N + 1):
            phase = 2 * math.pi * j * l / L
            sum_sin += y[j-1] * math.sin(phase)
        b_l = (2.0 / N) * sum_sin
        omega_l = 2 * math.sin(l * math.pi / (2 * (N + 1)))  # при k/m=1
        E[l-1] = 0.5 * omega_l**2 * b_l**2
    
    return E

2. Верификация алгоритмов

2.1. Проверка для гармонической цепочки

Для верификации проведён вычислительный эксперимент. Система из N = 31 частицы инициализирована первой модой (l = 1) с малой амплитудой (A = 0.01). Шаг интегрирования dt = 0.01. При α = 0 система должна вести себя как набор независимых гармонических осцилляторов.

Результаты:

  1. Сохранение энергии: полная энергия системы оставалась постоянной с точностью до 10⁻⁶% на протяжении 10⁵ шагов.
  2. Частота колебаний: измеренный период центральной частицы (i = 16) составил T_meas = 6.283, теоретический ω_theor = 1.0 (расхождение менее 0.01%).
  3. Отсутствие перекрёстных мод: ДПФ показал энергию других мод (l > 1) ниже уровня 10⁻¹⁰ от энергии основной моды.
